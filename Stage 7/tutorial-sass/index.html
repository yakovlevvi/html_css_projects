Знакомство с препроцесором Sass

Так что же такое CSS препроцессор? Это инструмент, который позволяет нам писать CSS код более удобно и эффективно. Он добавляет дополнительные возможности к обычному CSS, такие как использование переменных, вложенность селекторов, более удобные возможности использования функций и многое другое. Мы пишем код по новым более удобным правилам, а CSS препроцессор преобразует этот код в обычный и привычный нам CSS.
